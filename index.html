Instructions:

  • While guessing the letters, the dashes will automatically update with the correct letter guessed
  • Please press a letter on your keyboard. You have 10 chances if you type a wrong letter
  • If you mistakenly type the same letter more than once, it will not count towards your 10 tries
  • If you mistakenly type the same correct letter more than once, the game will let you know and allow you to type a different letter
  • Once you type 10 wrong letters, the game will be over and the correct answer will be displayed to you
  • Once you type the correct letter, the game will reset if you want to start again
